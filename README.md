# YouTube Video Manager (Python)

A simple CLI-based Python application to manage YouTube video information.
This project was built to practice core Python programming concepts and
basic application structure.

## Features
- List all videos
- Add a new video
- Update existing video details
- Delete videos
- Persistent storage using a JSON file

## Tech Stack
- Python
- JSON (for data storage)

## How It Works
The application runs in the terminal and provides a menu-driven interface.
User data is stored in a JSON file so that it persists even after the program
is closed.

## Purpose
This is a learning project focused on:
- Python fundamentals
- File handling
- CRUD operations
- Writing modular and maintainable code
